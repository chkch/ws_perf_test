# 基础使用
- ws-perf-test [URL] -c [总连接数] -b [批次连接数] -i [批次间隔(ms)] --data [自定义JSON文件路径] -d [持续时长(s)] 

- 多币对·单channel 
ws-perf-test ws://xxxxx/v1/public -d 300 -c 50000 -b 500 -i 1000 --data wss-1.json
ws-perf-test ws://xxxxx/v1/public -d 300 -c 10000 -b 200 -i 1000 --data wss-1.json
ws-perf-test ws://xxxxx/v1/public -d 300 -c 1500 -b 100 -i 2000 --data wss-1.json
ws-perf-test ws://xxxxx/v1/public -d 300 -c 800 -b 100 -i 2000 --data wss-1.json
ws-perf-test ws://xxxxx/v1/public -d 300 -c 200 -b 10 -i 2000 --data wss-1.json

- 多币对·多channel(6个) 
ws-perf-test ws://xxxxx/v1/public -d 300 -c 40 -b 5 -i 10000 --data wss-2k-2.json 
ws-perf-test ws://xxxxx/v1/public -d 300 -c 3 -b 1 -i 10000 --data wss-2k-3.json 

- 12小时稳定性压测 
nohup ws-perf-test ws://xxxxx/v1/public -c 1000 -b 100 -i 1000 --data wss-1.json -d 43200 -l stability_50k.log --export result.json --sharedTimers --killZombie > /dev/null 2>&1 & 

- 统计订阅数量 
grep -o 'channel' wss.json | wc -w
